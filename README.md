# RoboLang
### A Language for Children to learn programming

## Example
```
begin
    goto 100,100
    repeat 10
        left 10
        circle 20
    end
    repeat 10
        down 10
        circle 20
    end
    repeat 10
        right 10
        circle 20
    end
    repeat 10
        up 10
        circle 20
    end
end
```
